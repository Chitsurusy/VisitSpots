1. 准备素材
在你的文件夹里准备好以下图片：

首页大图：建议横向图片，命名为 banner.jpg。
地图底图：古村的平面图（如果没有平面图，就留空，系统会自动用灰色背景代替大头针位置），命名为 map.jpg。
印章图：5张图片（或网络链接），对应各个景点。
2. 修改配置 (你的“管理员后台”)
用记事本打开上面的 index.html，找到大约 150行 左右的 const CONFIG = { ... } 部分。

修改文字：

把 homeTitle 后面的引号内容改成你的标题。
把 homeDesc 改成你的宣传语。
修改图片：

把 homeBannerImg 改成 "banner.jpg"（假设你把图片放在同一个文件夹里）。
把 mapBackgroundImg 改成 "map.jpg"。
设置打卡点（地图位置）：
在 spots 数组里，你可以看到 x 和 y。

x 是横向位置（0-100），y 是纵向位置（0-100）。
例如 { x: 10, y: 20 } 表示这个点在地图左上角 10% 宽，20% 高的位置。
你可以先上传看看，然后在手机上打开，觉得位置不对，回来改数字，重新上传即可。

3. 上传 & 生成二维码
依然使用 Vercel 上传这个文件夹。
二维码逻辑不变，每个景点的二维码是：你的网址?spot=1 (数字对应 spots 里的 id)。

🗺️ 关于地图的特别说明
找一张古村的风景照做底图，然后通过修改 x 和 y 的数值，把红色的大头针放在大概正确的位置。
